markdown.tmbundle

by Fletcher T. Penney

Variation of the markdown TextMate bundle with improved support for MultiMarkdown

This is a fork of the original Markdown bundle for TextMate that includes proper support for MultiMarkdown, as well as some additional features for working with MMD documents.

Please see the main MultiMarkdown page for further information about MultiMarkdown, or the MultiMarkdown TextMate Bundle page for more information about this bundle.

Author

MultiMarkdown related changes by Fletcher T. Penney (owner@fletcherpenney.net). Based on the original Markdown bundle for TextMate by multiple authors.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running:

$ git clone git://github.com/fletcher/markdown.tmbundle
