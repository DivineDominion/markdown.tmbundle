# <a href="http://github.com/fletcher/markdown.tmbundle">markdown.tmbundle</a> #


## by <a href="http://github.com/fletcher">Fletcher T. Penney</a> ##

<div class="description">
  Variation of the markdown TextMate bundle with improved support for MultiMarkdown
</div>

 

This is a fork of the original Markdown bundle for TextMate that includes proper support for MultiMarkdown, as well as some additional features for working with MMD documents.

Please see the [main MultiMarkdown page](http://fletcherpenney.net/multimarkdown/) for further information about MultiMarkdown, or the [MultiMarkdown TextMate Bundle](http://fletcherpenney.net/multimarkdown/multimarkdown_bundle_for_textm/) page for more information about this bundle.


## Author ##

MultiMarkdown related changes by Fletcher T. Penney (<owner@fletcherpenney.net>).  Based on the original [Markdown bundle](http://github.com/textmate/markdown.tmbundle/tree/master) for TextMate by multiple authors.
